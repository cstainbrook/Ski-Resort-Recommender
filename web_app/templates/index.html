About Cooper's Chair Lift

Cooper's chair lifts is a ski resort recommender, which aggregates reviews from onthesnow.com to make user-specific recommendations. This was simply a project that I put together to practice my skills in the recommendation space.

Onthesnow.com is an online ski community and ski resort information provider, where users can find resorts, post reviews, and get information about the quality of the snow.

The recommendation engine is built using python and the graphlab package. This website is built using Flask and is hosted on an Amazon Web Services server.
